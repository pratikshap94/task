B.E. Project

Title: Development of Super capacitor using Nano materials.

Description: A Super-Capacitor using Zinc Oxide (ZnO) Nano Materials was developed. Five successful models were prepared and comparison of Activated Carbon dependence & ZnO Nano Material dependence experiments were conducted successfully. It was found that the Capacitance value depends on quality of material used such as activated carbon power, Electrode material & concentration of Nano Material. Team Size: 4


M.Tech Project

Title: Implementation of PLC Communication using Wireless Connectivity.

Description: This project aims at developing an IoT platform which allows a central server to communicate with all the different PLC controllers via bluetooth. It is implemented by connecting bluetooth connected arduino to PLC. The data transmitted by other bluetooth connected system will be received by the bluetooth connected to arduino and vice versa. According to code running on the arduino, the data will be sent to PLC. It is also possible to establish wireless communication between PLC and field devices by connecting field devices to the Bluetooth. Team Size: 1
